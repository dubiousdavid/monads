# monads

Monads: Reader, Writer, State, Either, and Maybe.

Transformers: Reader+Either, Reader+Maybe, Reader+Writer, Reader+Writer+Either,
Reader+Writer+Maybe, Writer+Either, Writer+Maybe, State+Either, State+Maybe.
